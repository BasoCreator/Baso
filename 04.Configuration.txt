< Configuration >

1. If you have already done configuration, you may need backup
    - Config.txt
    - ConfigFontName.txt

2. Get Internal Configuration Files

1) Click Menu : Etc > Make Config File

2) In your App directory, you can find files.
    - Config.txt
    - ConfigFontName.txt

3. Edit Configuration Files.

You can see default settings.
You can change and save.

1) RGB means Color

2) Etc
    - is_dark_mode : 1 = dark mode, 0 = white mode

4. Set Config

1) Click Menu : Etc > Set Config

2) Restart App

The screen does not redraw automatically.

You must close the window and run it again for it to execute with the 
configured values.

5. Recommanded Configuration

Download the '05.Config(Dark).txt' file for Dark Mode or the 
'06.Config(White).txt' file for White Mode.

Rename the downloaded file to 'Config.txt'.

6. If any troubles

Delete '~Config', '~FontName', '~WinPos' files.

It makes initial state.
