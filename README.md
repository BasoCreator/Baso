Baso(Basic Source Code Editor).

I have no plans to upload the source code. 

It is free to use, and it is also free for each individual to donate.

Baso is a secure and minimalist text editor that never uses the network and never collects any PC information.  
It is designed as an ASCII-only editor, focusing on simplicity, privacy, and lightweight performance.  
Originally derived from the commercial editor Amera, Baso is now distributed through GitHub Releases as an executable file, free for anyone to download and use.  

System Requirements
- CPU : Intel Processor
- OS : MS Windows 10 (64-bit /32-bit)
- App : 32-bit
